# 🚀 คู่มือ Deploy FishStock ใช้งานจริง

## ขั้นตอนที่ 1: จดโดเมน (Custom Domain)

### แนะนำผู้ให้บริการ
| ผู้ให้บริการ | ราคา/ปี | หมายเหตุ |
|---|---|---|
| **Cloudflare Registrar** | ~฿300 (.com) | ราคาทุน ไม่มีค่าใช้จ่ายซ่อน |
| **Namecheap** | ~฿400 (.com) | สมัครง่าย UI ดี |
| **GoDaddy** | ~฿500-700 (.com) | ใหญ่ มี support ไทย |
| **THNIC** | ~฿800 (.co.th) | สำหรับโดเมนไทย ต้องมีบริษัท |

### ชื่อโดเมนแนะนำ
- `fishstock.com`
- `ชื่อร้าน.com`
- `ร้านชื่อ-pos.com`

---

## ขั้นตอนที่ 2: ผูกโดเมนกับ Vercel

1. ไปที่ **Vercel Dashboard** → เลือก project `fishing`
2. **Settings → Domains**
3. กด **Add** → ใส่ชื่อโดเมนที่จดมา (เช่น `fishstock.com`)
4. Vercel จะแสดง DNS Records ที่ต้องตั้งค่า:
   ```
   A     @     76.76.21.21
   CNAME www   cname.vercel-dns.com
   ```
5. ไปที่หน้าจัดการ DNS ของผู้ให้บริการโดเมน → เพิ่ม records ตามข้างต้น
6. รอ propagate (5 นาที - 24 ชม.)
7. Vercel ตรวจสอบเสร็จ → **SSL อัตโนมัติ**

---

## ขั้นตอนที่ 3: แก้ Settings ในแอป

หลังโดเมนใช้งานได้ ไปที่:
**ตั้งค่าระบบ → ข้อมูลร้าน** → กรอกครบ:
- ชื่อร้าน
- ที่อยู่
- เบอร์โทร
- เลขผู้เสียภาษี (ถ้าจดทะเบียน)
- URL โลโก้

ข้อมูลนี้จะแสดงใน:
- หน้า login
- ใบเสร็จ
- รายงาน Z-Report

---

## ขั้นตอนที่ 4: เพิ่มผู้ใช้พนักงาน

**ตั้งค่าระบบ → จัดการผู้ใช้งาน**

1. กดเปลี่ยนรหัสผ่าน `admin` จาก `admin1234` เป็นรหัสที่ปลอดภัยกว่า
2. ลบ user `staff` ที่มาเป็น default
3. เพิ่ม staff จริงตามจำนวนพนักงาน
4. แต่ละคนใช้ username/password ของตัวเอง

⚠️ **อย่าให้พนักงานรู้รหัสผ่าน admin**

---

## ขั้นตอนที่ 5: ติดตั้งเป็น App บน iPad

### บน iPad (Safari)
1. เปิด `https://yourdomain.com` ใน Safari
2. กดปุ่ม **Share** (รูปสี่เหลี่ยมมีลูกศรขึ้น)
3. เลื่อนลง → **Add to Home Screen**
4. ตั้งชื่อ "FishStock" → Add

### บน Android (Chrome)
1. เปิด URL ใน Chrome
2. เมนู ⋮ → **ติดตั้งแอป**
3. ยืนยัน Install

### บน PC (Chrome/Edge)
1. เปิด URL
2. ดูที่ Address bar → จะมี icon ติดตั้ง
3. คลิกแล้ว Install

---

## ขั้นตอนที่ 6: ตั้งค่าการ Backup

### Auto Backup (Manual)
ทุกเย็นก่อนปิดร้าน:
1. **ตั้งค่าระบบ → สำรองข้อมูล** → กด **Export ข้อมูลทั้งหมด**
2. Save ไฟล์ JSON ลง Google Drive / Dropbox / USB

### Z-Report พิมพ์ออก
ทุกเย็น:
1. ไปเมนู **ปิดยอดประจำวัน (Z-Report)**
2. นับเงินในลิ้นชัก → กรอกในระบบ
3. กด **ปิดยอด/บันทึก**
4. กด **พิมพ์รายงาน** → เก็บใส่แฟ้ม

## ⚠️ ข้อจำกัดของระบบปัจจุบัน

| ข้อจำกัด | วิธีแก้ |
|---|---|
| ข้อมูลอยู่ใน Browser เครื่องเดียว | Backup JSON ทุกวัน + ใช้เครื่องเดียวเป็นหลัก |
| ใช้หลายเครื่องไม่ sync กัน | ใช้เครื่องเดียวเป็น "Master" / Backup ก่อนย้าย |
| ไม่มี Cloud auto-backup | Export JSON ทุกวัน |
| ล้าง browser data = ข้อมูลหาย | Backup ทุกวัน |
