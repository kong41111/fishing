# 🎣 FishStock — ระบบจัดการคลังสินค้าอุปกรณ์ตกปลา

ระบบบริหารจัดการคลังสินค้า, การขาย (POS), และรายงาน สำหรับร้านอุปกรณ์ตกปลา

## 🌐 Live Demo

👉 [https://your-app.vercel.app](https://your-app.vercel.app)

## ✨ Features

- 📊 แดชบอร์ดสรุปยอดขาย/สต็อก
- 📦 จัดการสินค้า + รูปภาพ
- 🛒 ระบบ POS (จุดขาย)
- 📥📤 รับ/จ่ายสินค้า
- 📈 รายงานสินค้าขายดี
- 🤝 จัดการซัพพลายเออร์

## 🚀 Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/fishstock.git
cd fishstock
# เปิด index.html ด้วยเบราว์เซอร์ หรือ
npx serve .
```

## 🛠 Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Canvas API (กราฟ)
- Unsplash (รูปสินค้า)
